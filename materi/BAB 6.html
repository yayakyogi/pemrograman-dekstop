Bab 6 Menggunakan Radiobutton



Setelah pada materi sebelumnya kita sudah mempelajari tentang listbox pada kesempatan kali ini kita akan belejar menggunakan toolbox Radiobutton. Radiobutton merupakan toolbox yang digunakan untuk memilih sesuatu seperti dibawah ini

  1. Langkah pertama adalah buat desain menggunakan groupbox, radiobutton dan button pada form serperti berikut


  2. Kemudian double klik button dan isi codingnya seperti dibawah ini


  3. Jalankan aplikasi dan akan menghasilkan output seperti berikut

    Ketika meilih lingkaran



  4. Cara diatas masih menggunakan button untuk menampilkan outputnya, lantas bagaimana bila kita ingin langsung menampilkan outputnya tanpa harus mengklik tombol button?

    Caranya adalah kita double klik terlebih dahulu masing-masing radio buttonnya, kemudian isi coding didalam prosedur radiobutton seperti dibawah ini


  5. Lalu jalankan aplikasi maka akan tampil output seperti dibawah ini

About

WEB ini dibangun dengan tujuan untuk digunakan sebagai media pembelajaran Pemrograman Dekstop. Didalam pembelajaran nanti akan dijelaskan apa saja yang perlu diperhatikan dalam membuat suatu aplikasi berbasis dekstop, mulai dari dasar penggunaan Text Editor, aplikasi tambahan seperti .NET Framework sampai dengan tools yang dibutuhkan untuk membuat suatu aplikasi berbasis Dekstop.

Tujuan akhir dari pembelajaran ini adalah agar siswa bisa mengetahui cara membuat aplikasi dekstop (GUI) dengan cara-cara yang sudah dijelaskan di pembelajaran ini, dan dipilihya media pembelajaran berbasis WEB ini adalah agar siswa bisa mengakses semua materi kapanpun, baik melalui laptop ataupun melalui smartphone masing-masing
