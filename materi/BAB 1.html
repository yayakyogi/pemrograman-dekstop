Bab 1 Pengenalan IDE untuk Pemrograman Dekstop



Pada materi ini membahas tentang dasar-dasar sebelum membuat program GUI dengan C#. Apa itu GUI? GUI (Graphical User Interface) Adalah anatrmuka pada sistem operasi atau komputer yang menggunakan menu grafis agar mempermudah para pengguna-nya berinteraksi dengan komputer atau sistem operasi. Jadi, GUI merupakan antarmuka pada sistem operasi komputer yang menggunakan menu grafis. Menu grafis ini maksudnya terdapat tampilan yang lebih nyaman menggunakan komputer. Menu grafis itu ya seperti ada garis-garis atau gambar-gambar dan tampilan yang tujuannya untuk memudahkan para pengguna menggunakan sistem operasi.


Dalam pengembangan aplikasi GUI C# dalam Pemrograman Dekstop ini menggunakan bantuan Sharpdevelop. Bahasa pemrograman C# hampir sama dengan bahasa C dan C++ sehingga mahasiswa tidak akan mengalami kesulitan yang berarti dalam pembelajar C#


  1. Untuk membuat sebuah Form GUI dengan C# terlebih dahulu buka SharpDevelop sehingga tampilan window seperti dibawah ini


  2. Klik New Solution atau File -> New -> Solution untuk membuat sebuah project baru. Sehingga akan tampil window seperti dibawah ini


  3. Pada categories C# -> Windos Application pilih template Windows Application. Setelah itu beri nama Project, contoh : LATIHAN. Pastikan nama Project tanpa spasi. Klik create untuk menampilkan lembar kerja project yang dibuat.


  4. Didalam file lembar kerja Mainform.cs terdapat dua buah tab yaitu Source dan Desain. Source digunakan untuk memasukkan perintah atau coding, sementara Design digunakan untuk membuat desain dari aplikasi


  5. Klik tab Tools -> Windows Forms untuk menampilkan tools dari desain yang akan dibuat. Window sebelah kiri merupakan Tools dari button dll, sementara window tengah merupakan form area tampilan daerah GUI, dan window kanan merupakan Properties yang berhubungan dengan Tools yang berada dalam desain form.

About

WEB ini dibangun dengan tujuan untuk digunakan sebagai media pembelajaran Pemrograman Dekstop. Didalam pembelajaran nanti akan dijelaskan apa saja yang perlu diperhatikan dalam membuat suatu aplikasi berbasis dekstop, mulai dari dasar penggunaan Text Editor, aplikasi tambahan seperti .NET Framework sampai dengan tools yang dibutuhkan untuk membuat suatu aplikasi berbasis Dekstop.

Tujuan akhir dari pembelajaran ini adalah agar siswa bisa mengetahui cara membuat aplikasi dekstop (GUI) dengan cara-cara yang sudah dijelaskan di pembelajaran ini, dan dipilihya media pembelajaran berbasis WEB ini adalah agar siswa bisa mengakses semua materi kapanpun, baik melalui laptop ataupun melalui smartphone masing-masing
