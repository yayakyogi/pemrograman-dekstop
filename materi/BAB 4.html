Bab 4 Menggunakan Combobox



Setelah sebelumnya kita mempelajari Cara mengambil nilai dari textbox pada materi selanjutnya ini kita akan membuat suatu Combobox pada form.

Apa itu combobox? Combobox adalah suatu toolbox yang biasa digunakan untuk membuat suatu pilihan yang berbentuk daftar seperti dibawah ini


Untuk membuat combobox seperti diatas ikuti langkah-langkah dibawah ini dengan seksama:

  1. Pertama buat project baru di sharpdevelop, kemudian klik Windows Forms lalu pilih combobox dan tambah label diatasnya dengan nama daftar warna seperti diatas.

  2. Kemudian untuk codingnya agar pilihan daftar didalam combobox bisa di lihat langsung pada saat di run pertama kali maka double klik pada mainform dan masukkan coding seperti dibawah ini


    Kemudian Run dan lihat hasilnya bila coding benar maka akan menunjukkan daftar di combobox seperti gambar diatas

  3. Untuk lebih memahami lagi materi ini kita coba tambahkan percabangan pada combobx ini, coba kalian tambahkan satu textbox disampingnya dan beri label baru dengan nama translate seperti dibawah ini

  4. Kemudian untuk codingnya agar ketika dipilih warna merah bisa langsung di translate Red di textbox, double klik combobox 1 lalu isi codingnya seperti dibawah ini


    Lalu run dan coba pilih warna yang lainnya, bila sukses maka pada textbox translate akan menampilkan arti dari warna yang dipilih.

Latihan

Buatlah 2 Combobox dan isi dari combobox pertama adalah Jurusan dan Mata Kuliah. Sehingga ketika dipilih jurusan maka pada combobox 2 akan menampilkan daftar jurusan dan ketika dipilih matkul maka akan muncul daftar mata kuliah seperti dibawah ini


Note : Gunakan code comboBox2.Items.Clear untuk mengosongkan combobox yang memiliki list.


Terimakasih :)

About

WEB ini dibangun dengan tujuan untuk digunakan sebagai media pembelajaran Pemrograman Dekstop. Didalam pembelajaran nanti akan dijelaskan apa saja yang perlu diperhatikan dalam membuat suatu aplikasi berbasis dekstop, mulai dari dasar penggunaan Text Editor, aplikasi tambahan seperti .NET Framework sampai dengan tools yang dibutuhkan untuk membuat suatu aplikasi berbasis Dekstop.

Tujuan akhir dari pembelajaran ini adalah agar siswa bisa mengetahui cara membuat aplikasi dekstop (GUI) dengan cara-cara yang sudah dijelaskan di pembelajaran ini, dan dipilihya media pembelajaran berbasis WEB ini adalah agar siswa bisa mengakses semua materi kapanpun, baik melalui laptop ataupun melalui smartphone masing-masing
