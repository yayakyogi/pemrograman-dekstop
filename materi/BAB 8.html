  • Aplikasi Sharpdevelop atau sejenisnya seperti Visual Studio
  • Xampp
  • Mysql Connector 6.9.8

About

WEB ini dibangun dengan tujuan untuk digunakan sebagai media pembelajaran Pemrograman Dekstop.
